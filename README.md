# Stunting-Risk-Compass
Interactive data visualization platform for mapping stunting risks using Python and Plotly to support data-driven public health policies.

📍 Stunting Risk Compass
Sebuah platform visualisasi data interaktif yang dirancang untuk memetakan dan menganalisis risiko stunting. Proyek ini bertujuan untuk mengubah data kesehatan yang kompleks menjadi wawasan visual yang mudah dipahami oleh pengambil kebijakan.

Fitur Utama:

Interactive Visuals: Menggunakan Plotly untuk grafik dan peta yang dinamis.

Risk Mapping: Identifikasi area prioritas untuk intervensi kesehatan berdasarkan indikator risiko.

User-Centric Design: Dirancang dengan fokus pada kemudahan navigasi data bagi pengguna non-teknis.

Tech Stack:

Python (Pandas, NumPy)

Plotly (Express & Graph Objects)
